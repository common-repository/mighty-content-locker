== Installation ==

1. Upload the `mighty-content-locker` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to the <strong>Content Locker</strong> settings page in the admin menu, and enter Your Content Locker URL produced by Mighty Content Locker
